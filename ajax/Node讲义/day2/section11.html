Node.js讲义

ES6中的新语法

1.const 一般用来定义常量 

// 常量就是不会经常发生变化的数据
const num = 20; // 定义了一定要赋值,不赋值也会报错,也会有块级作用域
// num = 30 // 不可以在下面重新被赋值,一般用于内置模块或是第三方模块的返回值的声明
console.log(num);

2.let 有块级作用域,没有变量的提升,可以重新赋值

console.log(i);
for(var i=0;i<3;i++){
  console.log(i);
}
console.log(i);

//换成如下再试一下:
console.log(i); // 没有变量的提升问题,变量只能在大括号里面使用
for(let i=0;i<3;i++){
  console.log(i);
}
console.log(i);

3.startsWith与endsWidth

var str ="abcdef"
console.log(str.startsWith('abc'));// 判断字符串中以谁开头
console.log(str.endsWith('f')); // 判断字符串是以谁结尾
