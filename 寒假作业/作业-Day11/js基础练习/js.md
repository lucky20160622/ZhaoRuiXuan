4.  使用一个 for 循环实现：1-100 之间所有偶数的打印
    要求：使用两种方式实现（其中一个方式必须使用 `continue` 关键字）

```js
// 第一种：
for (var i = 1; i <= 100; i++) {
  if (i % 2 == 0) {
    console.log(i);
  }
}

// 第二种：

for (var i = 1; i <= 100; i++) {
  if (i % 2 != 0) {
    continue;
  }
  console.log(i);
}
```
