## js编程题
1. 打开网页时弹出：全栈牛逼！！！(只有确定键)

    ```html
    alert('全栈牛逼！！！')
    ```

2. 让用户可以在网页输入年龄

    ```html
    var prompt('请输入您的年龄')
    ```

3. 打开网页弹出：您打开了一个网页，是否继续？（有确定键和取消键）

    ```html
    confirm('您打开了一个网页，是否继续？')
    ```

4. 让用户输入一个数字，实现判断该数字是小数还是整数并告诉用户

    ```js
    var num=prompt('请输入一个数字')
    if(num==parseInt(num)){
        alert('您输入的数字是整数')
    }else{
        alert('您输入的数字是小数')
    }
    ```
